# tg bot и серверная часть

## Телеграм-бот
- Раз в час проверяет объявления.
- Если объявления подходят под условия, отправляет пользователю информацию о новом нужном товаре.

### Настройки бота
- В боте можно будет выбрать параметры для отбора товаров (например, название, тип изношенности и т.п.).

---

## Сервер
### База данных (`bd`)
- **items** — товары, которые нужно отслеживать.
- **tested_items** — товары, которые уже отправлены пользователю.

### Парсеры
#### Selenium парсер
- Вручную вводятся название товара из `items` и тип изношенности.
- Запоминает ссылку и количество страниц.
- После выполнения возвращает список вида: `[[ссылка, кол-во страниц], ...]`.

#### aiohttp парсер
- Проходит по сохранённым ссылкам и страницам.
- Собирает актуальные цены.
- Добавляет нужные ссылки в `tested_items`.
- Возвращает данные о ценах по запросу в Telegram-бот.
